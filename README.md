# Object-Oriented Programming Laboratory

This repository contains practical implementations of Object-Oriented Programming (OOP) concepts in Java, progressing from basic syntax to advanced system design, design patterns, and database connectivity.

## Laboratory Assignments

* **Lab 1**: Introduction to Java programming, basic control structures, and array manipulation.

* **Lab 2**: Implementation of core OOP principles including Classes, Objects, and Constructors.

* **Lab 3**: Exploration of Inheritance, Method Overriding, and Package management.

* **Lab 4**: Error handling via Exceptions and File I/O operations (reading from and writing to text files).

* **Lab 5**: Utilization of advanced OOP features such as Abstract Classes, Interfaces, and Polymorphism.

* **Lab 6**: Implementation of custom Data Structures (Stack, Set) and Introduction to Generics.

* **Lab 7**: Multithreading, Thread synchronization, and Concurrency management in Java.

* **Lab 8**: Object-Oriented System Design and Modeling for complex real-world scenarios using Design Patterns (e.g., Flight Management, Parking Garage).

* **Lab 9**: Advanced Java problem-solving and Unit Testing using the JUnit framework.

* **Lab 10**: Database integration and relational data management using Java Database Connectivity (JDBC).

* **Lab 11**: Application of Software Design Patterns (specifically the Observer Pattern) and Event-Driven Architecture.

## Compilation and Running Guide

Kindly refer to the pdf report files in each Lab folder to see how to compile and run the code files

### Utility Scripts

This repository includes a few helper shell scripts in the root directory to streamline development and submission:

    ./run: A quick-execution script to compile and immediately run a provided Java file in one step.

    ./clearClass: A cleanup utility that recursively finds and removes all compiled .class files to keep the workspace clean.

    ./createZip: A packaging script that bundles the necessary source files and PDF reports into a .zip archive for easy assignment submission.
